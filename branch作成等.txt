Git ブランチ作成手順

git branch ブランチ名
→ブランチ作成

git branch -aまたはgit branch --allで
全ブランチの確認が出来る

git checkout ブランチ名
ブランチの移動

・ブランチの削除
 git branch -d ブランチ名
